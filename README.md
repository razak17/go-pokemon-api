# go-pokemon-api

Simple Go REST API

This repo contains a simple/basic HTTP server in Go.

- net/http package to start and serve HTTP server
- [Chi](https://github.com/go-chi/chi) to handle routes

## Build the app

```bash
go build -o bin/go-pokemon-api main.go
```

## Run the app

```bash
./bin/go-pokemon-api
```
